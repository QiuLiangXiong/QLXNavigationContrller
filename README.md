# QLXNavigationController
 用法和 UINavigationController 完全一致 可以当做UINavigationController
 
 1.QLXNavigationController 解决了不同导航栏背景不一致带来的过度效果不佳问题。  
 
 2.支持push一个导航栏控制器。
